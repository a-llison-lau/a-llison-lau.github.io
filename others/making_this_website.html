Making This Website

The map of front end developing…

How can a computer science student not have his/her own website! One day I decided to build my own website while actually learning web development. This website is not just for my potential supervisors and employers but also for myself, hence this password protected section of my website! But it is still not password protected because I’m trying! But never mind because no one will suddenly discover this website anyway… Unless you are my CSC207 group mate and you are stalking me 👀.

Now back to building this website. I like doing design, so I shall start with designing. This starts with drawing the site map, where we plan what pages are in the website, then wire framing the website, which is sketching its skeleton. Then add details to the design, such as color and content. Did this with Figma! It is very intuitive to learn, very intelligent and have great features. Actually there’s more to designing a website. That’s why people need UI designers. Colour psychology and stuff, but it’s my own website so whatever.

(Because whatever so I didn’t follow the design at all, because it is too difficult)

Then learn HTML / CSS / Javascript and translate everything from Figma design to code. Separating design from coding is painful but necessary if you want a good design. But it is really painful! It took me so long to figure out just the position of one division. Choosing a good YouTube video really saves you tons of time! But finally I have one website of my own, although I might abandon this website later.

There are still a lot of features I am planning to add to this website, including customizing the cursor, image magnifying effect, transition effects, etc. Come back from time to time and see if I added anything!

Now that I know the basics of making a website, it should take way shorter time to build another one. If you read until here and you want a personal website or a website for your club or society, I can build one for you ;)

And a side note. I started late as a software developer, and I wouldn't say I have talent in programming. But to anyone who think they are bad at programming, just start doing it!
